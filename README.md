# NotesApp

Приложение для создания заметок

## Реализованный функционал
- Создание, просмотр, редактирование и удаление заметок.
- Хранение заметок в Room (данные сохраняются после перезапуска приложения).
- Поиск по заголовку.
- Теги у заметок + фильтрация по тегам на главном экране.
- Добавление изображения в заметку.
- Сохранение черновика новой заметки, если выйти назад без кнопки "Сохранить".

## Сборка и запуск
1. Убедитесь, что установлен Android Studio (или Android SDK + JDK 11).
2. Соберите проект:
   ```bash
   ./gradlew assembleDebug
   ```
3. Запустите на эмуляторе/устройстве:
   ```bash
   ./gradlew installDebug
   ```

## Используемые библиотеки
- **Jetpack Compose + Material 3** — декларативный UI.
- **Navigation Compose** — навигация между экранами.
- **Lifecycle ViewModel + Runtime Compose** — управление UI-состоянием в MVVM.
- **Room** — локальная база данных для заметок.
- **DataStore Preferences** — хранение черновика заметки.
- **Coil Compose** — отображение выбранных изображений по URI.

## Архитектура
Приложение разделено на слои `data / domain / presentation` и использует MVVM.
